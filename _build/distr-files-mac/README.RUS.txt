JA2-Stracciatella для Mac OS X

Распакуйте архив с игрой.

JA2-Stracciatella может работать с разными версиями оригинальной игры.  Чтобы
начать игру, запустите command файл, соответсвующий вашей установленной
версии.  Так, если у вас установлена английская версия игры, то следует
запускать ja2-ENGLISH.command.  Если установлена "Jagged Alliance 2.  Агония
власти", то следует запускать ja2-RUSSIAN.command.  Если установлена "Jagged
Alliance 2. Золотая серия", то ja2-RUSSIAN_GOLD.command.

После того как вы в первый раз запустите игру, в вашем домашнем каталоге
("/Users/ВашеИмя") появится скрытая папка ".ja2".  В ней есть файл "ja2.ini".
Отредактируйте этот файл, прописав в нем правильный путь к файлам данных
Jagged Alliance 2 в строке "data_dir = [...]".  Назовем папку с игровыми
данными $DIR, тогда в ней должно быть следующее:
- $DIR/Data/*.slf           - все файлы .slf с инсталляционного И игрового CD
- $DIR/Data/TILECACHE/*.sti - все файлы из директории TILECACHE инсталлированной
- $DIR/Data/TILECACHE/*.jsd   игры

Имеется возможность изменить видео разрешение игры с помощью ключа '-res'.
Так например, чтобы запустить игру в режиме 1024x768, выполните следующую
команду:

  ja2 -res 1024x768

Любая разумная комбинация ширины и высоты должна работать.  Можете
поэкспериментировать и подобрать ту, которая подходит вам лучше всего.

JA2-Stracciatella также доступна для Windows и Linux.
